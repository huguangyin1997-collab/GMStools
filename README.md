GMStools 是一款专为提升工作效率而设计的综合性桌面工具软件，集成了多个实用功能模块，旨在帮助用户轻松处理日常技术任务。无论您是测试工程师、开发人员还是项目管理者，GMStools 都能为您提供便捷的解决方案。

主要功能
检查报告：快速生成和查看各类测试报告，支持自定义模板。

DB工具：简洁易用的数据库管理界面，支持常见数据库的查询与维护。

模块对比：对比不同模块的配置或代码差异，一目了然。

SMR对比：专门针对 SMR（软件需求规格）的对比分析工具。

CV自动化：自动生成或整理简历信息，提高招聘效率。

免责声明与关于：内置详细的免责声明和开发团队信息，保障用户权益。

技术特点
基于 PyQt6 开发，界面美观，跨平台运行。

采用自定义无边框窗口设计，支持拖拽、最大化/最小化等操作。

支持背景图片自定义，提供初音未来主题等视觉元素。

单实例运行保护，避免重复启动。

配置文件加密存储用户同意状态，兼顾体验与合规。

适用场景
GMStools 适用于软件测试、开发文档管理、数据处理等多种场景，尤其适合需要频繁进行报告生成、数据对比的团队和个人。
